Realms Plus adalah langganan untuk konten dan server pribadi Realm untuk digunakan dengan versi Minecraft yang didukung (permainan dijual terpisah) di "Playstation:nbsp:4" dan platform lain yang didukung, tempat kamu memiliki permainan (Minecraft Java Edition tidak didukung); langganan aktif diperlukan untuk menggunakan konten Realms Plus. Konten selalu bervariasi dan mungkin tidak tersedia di semua negara. Langganan "Playstation:nbsp:Plus" diperlukan untuk main di "Playstation:nbsp:4" (dijual terpisah). Jika langgananmu berhenti atau item konten dihapus dari katalog, kamu harus mengaktifkan ulang langganan atau membeli konten secara terpisah agar bisa terus menggunakannya. Jika langganan tidak aktif selama lebih dari 18 bulan, data Realm kamu akan dihapus dari server. Biaya ISP mungkin berlaku. Diskon Realms Plus tidak bisa digabungkan dengan penawaran lain dan tidak bisa ditukar dengan uang tunai; Persyaratan, fitur, layanan online, atau langganan dapat berubah dan/atau berakhir. Sesuai dengan Perjanjian Layanan Microsoft (microsoft.com/msa) dan Perjanjian Lisensi Pengguna Akhir Realms (https://www.minecraft.net/en-us/realms/terms/).
