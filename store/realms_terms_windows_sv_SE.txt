Realms Plus är ett abonnemang på innehåll och en personlig Realms-server för användning i Minecraft-versioner som stöds (spelet säljs separat) på Windows och andra plattformar som stöds där du äger spelet (Minecraft Java Edition stöds inte); ett aktivt abonnemang krävs för att använda Realms Plus-innehåll. Om abonnemanget sägs upp eller om innehåll tas bort från katalogen måste du återuppta abonnemanget eller köpa innehållet separat för att fortsätta använda det. Om abonnemanget är inaktivt i över 18 månader kommer dina Realms-data att tas bort från servern. Internetleverantörens avgifter kan tillkomma. Realms Plus-rabatter kan inte kombineras med andra erbjudanden och kan inte bytas ut mot pengar; krav, funktioner, onlinetjänster och abonnemanget kan komma att ändras eller läggas ner. Omfattas av avtalet för Microsoft-tjänster (microsoft.com/msa) och slutanvändarlicensavtalet för Realms (https://www.minecraft.net/sv-se/realms/terms/).
